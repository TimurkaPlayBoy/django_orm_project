# School Schedule System

Система керування шкільним розкладом

## пис проекту
Система для управління предметами, вчителями, учнями, класами та розкладом занять.

## Технології
- Backend: Flask + SQLAlchemy
- Database: SQLite
- Frontend: HTML, CSS, JavaScript

## оделі
- Uroks (предмети)
- Teacher (вчителі)
- Class (класи)
- Student (учні)
- Schedule (розклад)
- Grade (оцінки)
